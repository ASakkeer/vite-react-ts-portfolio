# Projects Page

## Purpose

Showcase portfolio projects with mock data.

## Components

- **ProjectsPage** – Full-page container
- **ProjectsList** – Grid of ProjectCard; optional `limit` for Home preview
- **ProjectCard** – Image, title, category, description

## Data

- Source: `@/data/projects.mock`
- Fields: id, title, category, description, image, link
- Images: Unsplash placeholders

- Replace mock data with real projects
- Add category filter
