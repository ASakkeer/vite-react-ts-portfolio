# Contact Page

## Purpose

Contact form and contact details display.

## Components

- **ContactPage** – Form (name, email, message) + contact details (email, phone, location, socials)

## Form

- React Hook Form + Zod validation
- EmailJS integration (requires VITE_EMAILJS_* env vars)

## Data

- Source: `@/data/contact.data`
- Replace placeholders with real email, phone, location, social URLs
